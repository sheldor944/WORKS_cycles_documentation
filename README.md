>
> **Steps:**
> 1. Download and install [DBeaver](https://dbeaver.io/)
> 2. Open DBeaver and create a new SQLite connection
> 3. Upload the `works_cycles.sqlite` file
> 4. Right-click on the database → **View Diagram**
> 5. That's it! You'll get a fully interactive ER diagram.
